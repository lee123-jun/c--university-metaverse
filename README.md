# 메타버스(Metaverse)
- 	가상 현실과 증강 현실을 포함한 몰입형 어플리케이션은 4차산업혁명 시대에 등장할 가장 흥미롭고 새로운 매체이다. Metaverse 구축을 위해서는몰입형 가상 세계를 만들기 위한 게임 엔진을 알고 있어야 한다. 이 과정에서 학생들은 헤드 마운트 디스플레이용 블루프린트를 사용하여 언리얼 엔진 4에서 처음부터 기본적인 VR 역학을 구축하는 방법을 배운다.
